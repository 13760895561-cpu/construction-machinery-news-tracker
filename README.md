# 工程机械情报台

这是一个面向工程机械行业的 RSS/栏目页聚合站点，覆盖两个板块：

- 行业信息：中央政策、重点省份政策、协会/统计局/海关/行业媒体数据。
- 主机厂信息：三一重工、中联重科、徐工机械、柳工、山推股份、山河智能、恒立液压、艾迪精密、浙江鼎力、三一国际、中国龙工、潍柴动力、中集车辆、安徽合力、杭叉集团。

## 功能

- Topic 入口：点击 Topic 可跳转到对应官方栏目或公司公告主页。
- 时间收纳：支持按日、周、月、季度、年度和全部历史回看。
- 数据归档：自动识别销量、进出口、开工率、作业小时、库存、均价、成本、投资等量化指标。
- 每小时更新：GitHub Actions 每小时运行一次采集脚本，并发布到 GitHub Pages。

## 本地运行

```bash
npm install
npm run update
npm run dev
```

## 发布

1. 在 GitHub 创建仓库并推送本目录。
2. 在仓库 Settings → Pages 中选择 GitHub Actions。
3. 手动运行一次 `Update and publish` workflow，之后它会每小时自动刷新。
4. GitHub Pages 地址会显示在 workflow 的部署结果中。
